5日贵州行程
第1天: 重庆 → 贵阳
中午 从重庆乘坐高铁前往贵阳(约2-2.5小时)
下午3:15-3:45 贵阳北站 → 贵阳铂尔曼酒店（16公里，乘坐出租车或网约车）
下午4:00 在贵阳铂尔曼酒店办理入住
下午5:00-5:30 贵阳铂尔曼酒店 → 东山寺（11公里，乘坐出租车或网约车）
下午5:30-7:00 游览东山寺，观赏晚霞
晚上7:00-7:15 东山寺 → 老凯里酸汤鱼(喷水池店)（4.5公里，乘坐出租车或网约车）
晚上7:15-8:45 在老凯里酸汤鱼享用晚餐，品尝贵州特色酸汤鱼
晚上8:45-9:15 老凯里酸汤鱼 → 贵阳铂尔曼酒店（7公里，乘坐出租车或网约车）
第2天: 贵阳 → 荔波(小七孔)
早上7:00 在贵阳铂尔曼酒店退房
早上7:30 从贵阳铂尔曼酒店出发自驾前往荔波（约4小时，280公里）
中午11:30 到达荔波，在当地餐厅用午餐
下午1:00 前往小七孔景区（距离荔波县城约25公里，车程约30分钟）
下午1:30-6:30 游览小七孔景区(约5小时)
晚上7:00 返回荔波县城
晚上7:30 在荔波县城用晚餐，品尝当地特色美食
晚上8:30 入住荔波酒店
第3天: 荔波 → 平塘(天眼) → 安顺
早上7:00 从荔波出发前往平塘(约2.5小时,150公里)
上午10:00-12:30 参观中国天眼科普基地(FAST)(约2.5小时)
下午1:00 在平塘用午餐
下午2:00 驱车前往安顺(约2.5小时,160公里)
晚上5:00 抵达安顺，入住酒店
晚上7:00-8:00 游览安顺古城墙(1小时)
晚上8:30 在安顺市区享用晚餐，品尝当地特色美食
第4天: 安顺(黄果树瀑布) → 青岩古镇
早上7:00 从安顺出发前往黄果树瀑布(约1小时)
上午8:00-下午3:00 游览黄果树瀑布景区(约7小时)
下午3:30 驱车前往青岩古镇(约1.5小时,90公里)
晚上5:00 抵达青岩古镇，入住古镇内的特色民宿
晚上6:00-8:00 漫步青岩古镇，感受明清时期的建筑风格，品尝当地特色小吃
第5天: 青岩古镇 → 贵阳 → 离开
早上8:00 在青岩古镇用早餐，继续游览古镇
上午10:00 从青岩古镇出发前往贵州省地质博物馆(约30分钟,20公里)
上午10:30-12:30 参观贵州省地质博物馆(约2小时)
中午12:30-1:30 在博物馆附近用午餐
下午 根据航班时间前往贵阳龙洞堡国际机场
